## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

```
Git Github
VS Code
NodeJS
```

### Setup

```sh
cd my-folder
git clone https://github.com/ghreza-crypto/space-travelers-hub
cd space-travelers-hub
```

### Install

```
npm install
```

### Usage

```
npm run build
npm start
```

### Run tests

```
npm test
```
